# MakeMoney TODO

这个 TODO 按当前使用场景排序：3 万资金、0 持仓、理财知识边做边学、长期目标年化 5%+。

原则：

- 先做新手保护，再做收益增强。
- 先把风险讲清楚，再让用户确认动作。
- 工具只做辅助决策和复盘，不自动下单。
- 5% 是长期目标刻度，不是收益承诺。

  - 每一笔显示：计划金额、估算份额、为什么买、如果下跌怎么办。
  - 未完成上一笔复盘时，不提示下一笔。

- [x] 交易前确认清单
  - 对每条可手动确认动作，要求用户回答：
    - 我是否理解这个 ETF 跟踪什么？

  - 估值缺失时提示风险，权益类动作需要额外确认。
  - 单周交易上限、单笔下限、单品种上限都要在 UI 里明确展示。
