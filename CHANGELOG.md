# 更新日志 / Changelog

本项目为个人 ETF 配置决策驾驶舱（教育/辅助工具，不构成投资建议、不自动下单）。
下面按批次记录重要改动。重要改动后建议在 git 打一个 tag（如 `v0.3`）。

## 未发布（当前工作区）

### 引擎与评审改进
- 新增回归测试 `engine/tests/test_engine.py`（stdlib unittest，无网络，50 个用例）：配置/策略校验、数据分级闸门、份额取整、首次建仓分批、压力测试、月度复盘、折溢价/规模分级、成交后持仓草稿。运行：`python3 engine/tests/test_engine.py`。
- 月度复盘 `reports.monthly_review()` + `/api/review/monthly` + 前端面板：按月看“是否守规则”（执行有无建议来源/数据质量/是否先看周报），不算收益率。
- 偏离复盘：月度卡新增“计划投入 vs 实际成交 vs 偏离”。
- 风险预算压力测试按资产类别拆解（`risk_budget.stress_contributions`，前端贡献条）。
- 观察池学习系统：`engine/learning.py` + `engine/learning_cards.yaml` + `/api/watchlist/learning(+/ack)`；学完 + 累计观察 4 次才解锁“可讨论纳入”，**始终不可直接买入**。
- AI 风险旗标富渲染（方向着色/置信度/影响资产/actionable/来源链接，仅放行 http(s)）；`flags_schema` 增加可选 `source_url`，`validate_flags` 校验。
- ETF 折溢价/清盘提示：`/api/etf/quality` 用实时 IOPV 自算溢价（QDII/黄金/货币阈值更严，≥1.5% 即“不足”），按总市值估规模/清盘风险；数据不可用如实标注，不伪造。
- 成交后持仓草稿 `/api/portfolio/draft`：据最近一次执行记录推算成交后持仓/现金，供手动核对填入，**绝不自动改持仓**。
- 启动体验：`app.py` 启动前自检依赖（缺失给安装提示），打印含端口与“改代码需重启”的横幅。

### 前端重构（精简化）
- 把约 18 个平铺板块重构成“顶部常驻决策区 + 5 个标签页”（行情与质量 / 我的组合 / 复盘与历史 / 观察池 / 回测）。
- ETF 曲线与质量合并为每只一张统一卡（两段渲染：曲线先出、质量随后补）。
- 周报“列表 + 详情”合并为主从视图；总览 KPI 与数据健康合并为状态条 + 详情弹层。
- 术语速查改为内联悬停 tooltip；工作台说明改为右下角悬浮帮助按钮。
- 慢加载（行情质量/回测）改为按需加载，首屏决策区秒出。
- 不引入构建工具/框架：仍是单文件、`python3 engine/app.py` 即用。
- 新增“后端过期自检”横幅：若在旧后端上跑新前端（Flask 不热重载），提示重启。
